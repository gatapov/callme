Инструкция по установке скрипта. Перед установкой зарегистрируйтесь на сайте sms.ru
Скрипт использует библиотеку jQuery 2.1.4

1.Скачайте архив и распакуйте в корневой каталог сайта
2.подключите скрипт callme.js на всех страницах, где необходимо установить обратный звонок

<script src="/callme/js/callme.js" type="text/javascript"></script>

3.Настройте свои данные в файле config.php

$adminemail - e-mail админа 
$backurl - На какую страничку переходит после отправки письма 

 $key -ваш api_id на сайте sms.ru
 $num -ваш номер в международном формате без "+"

4.Интервал дрожания значка настраивается в файле callme/tpl/form.html в строке:


setInterval(function() {$( ".callmeicon" ).effect( "shake" );}, 5000);

время в мс.

5. Чтобы какой-либо элемент вызывал скрипт присвойте ему id="callmeicon"
